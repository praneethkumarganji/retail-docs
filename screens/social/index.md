 # Social — Content & Publishing (Overview)

This folder contains the Social screen and detailed feature documentation.

ASCII Layout (detailed)

```
+-----------------------------------------------------------------------------------------------+
| HEADER: [Logo] [Create Post] [Schedule] [Calendar] [Queue] [Search posts] [Notifications] [User] |
+-----------------------------------------------------------------------------------------------+
| NAV (left)                                | MAIN: Composer / Calendar / Workspace                |
| ----------------------------------------- | ---------------------------------------------------- |
| [Dashboard]                               | TOP: Composer Toolbar (channel select, audience)     |
| [Products]                                | ---------------------------------------------------- |
| [Social]                                  | LEFT: Composer Panel                                  |
| [Assets]                                  |  - fields: text, media uploads, captions, hashtags    |
| [Analytics]                               |  - product tagging, preview, scheduling options       |
| [Inbox]                                   | ---------------------------------------------------- |
| [Settings]                                | MIDDLE: Calendar / Queue view                         |
|                                           |  - calendar with scheduled posts, drag & drop         |
|                                           |  - queue list with reorder and bulk actions           |
|                                           | ---------------------------------------------------- |
|                                           | RIGHT: Insights & Inbox                                |
|                                           |  - Post Analytics: impressions, engagements, clicks   |
|                                           |  - Inbox: comments, direct messages, assign/resolve   |
|                                           | ---------------------------------------------------- |
|                                           | BOTTOM: Approval & Activity Panel                     |
|                                           |  - approval queue, comments, version history          |
+-----------------------------------------------------------------------------------------------+
| RIGHT RAIL: Quick Actions | Help Articles | Rate Limit Status | Support Chat                |
+-----------------------------------------------------------------------------------------------+
| FOOTER: publishing status, connected channels, account limits                                   |
+-----------------------------------------------------------------------------------------------+
```
