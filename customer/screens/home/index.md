# Home — Storefront Landing (Overview)

This folder contains the customer Home (storefront landing) screen spec and detailed feature documentation.

ASCII Layout (desktop)

```
------------------------------------------------------------------------------------+
| HEADER: [Brand logo] [Main nav] [Search] [Account] [Wishlist] [Cart ▾]           |
------------------------------------------------------------------------------------+
| HERO: primary campaign banner (image/video, headline, CTA)                       |
------------------------------------------------------------------------------------+
| SECONDARY PROMOS: smaller banners (sales, new arrivals, seasonal)                |
------------------------------------------------------------------------------------+
| FEATURED CATEGORIES: icon tiles linking to key collections                        |
------------------------------------------------------------------------------------+
| FEATURED PRODUCTS: carousel/grid (bestsellers, new in, curated picks)            |
------------------------------------------------------------------------------------+
| PERSONALIZED STRIP (if logged in): "Picked for you"                              |
------------------------------------------------------------------------------------+
| FOOTER: help links, policies, about, language/currency switcher                  |
------------------------------------------------------------------------------------+
```
